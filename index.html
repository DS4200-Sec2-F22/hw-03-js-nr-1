hw-03: Interactive Webpage, and Marks & Channels

VISUALIZATION 1

Marks:

  • Areas, representing states;
  • Lines, representing state borders;
  • Lines, representing electoral vote count
Channels:
  • Color hue represents the candidate that each state voted for (Clinton vs. Trump vs. mixed votes cast). This is effective because blue and red are easily discriminable, as well as having established meanings.
  • Size is also a channel, since the length of the blue and red lines above the map represent the number of would-be votes cast for each candidate. This is effective because users will be able to accurately tell which bar is longer, due to the arrow at the midpoint.

VISUALIZATION 2

Marks:

  • Points, representing an aggregation of similar batted balls (given two attributes);
  • Lines, representing separation between types of hits (e.g. fly balls vs. line drives);
  • Areas of color, representing similarly-effective batted balls
Channels:
  • Color hue and saturation represent scoring value of each ball. Hue represents whether a ball is below or above average, while saturation represents the magnitude of that value. This is effective because scoring value is divergent, so a divergent colormap works (see Colormap(s))
  • Horizontal position represents the exit velocity of the ball, while vertical position represents the angle of a hit. This is effective because the visualization creates a 2D grid that plots items by these two attributes, helping demonstrate their interactions.

Visualization 1 uses a categorical colormap with two colors (red and blue), which represent Republican Donald Trump vs. Democrat Hillary Clinton, respectively.
This was an effective choice of colormap.

  • Since blue and red are already widely used for Democrat and Republican candidates, viewers will already associate the colors with the intended attributes.
  • The categorical (either/or) nature of the votes-cast attribute makes a categorical colormap a natural choice.

Visualization 2 uses a divergent colormap to represent scoring value of a batted ball, with pink representing below-average and orange representing above-average. This colormap is an effective choice.
  • Relation to an established average is an ordered, divergent attribute, so a divergent colormap fits well.
